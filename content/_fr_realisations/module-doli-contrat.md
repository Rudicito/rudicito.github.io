Ce module [Dolibarr](https://www.dolibarr.org/) aide les consultants à gérer les contrats et à suivre l'état des factures, dont les factures de situation.
Il regroupe les informations importantes pour éviter les oublis.

Le projet utilise une interface simple pour consulter les contrats, les échéances et les informations de
facturation.

Le module est developpé en PHP (le langage de Dolibarr).
