Ce projet est un site de commerce en ligne développé en PHP.
Il permet de consulter un catalogue de produits informatique, d'ajouter des articles dans un panier, puis de passer commande.

Il est développé en PHP et utilise ces technologies :

- [Symfony](https://en.wikipedia.org/wiki/Symfony) : un framework pour construire des applications Web.

- [Bootstrap](https://en.wikipedia.org/wiki/Bootstrap_(front-end_framework)) : un framework de mise en page
  pour avoir une interface jolie et responsive.

- [MariaDB](https://en.wikipedia.org/wiki/MariaDB) : une base de données relationnelle pour stocker les
  comptes, les produits et les commandes.
