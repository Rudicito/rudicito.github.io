Horizon est un site de gestion de médiathèque.
Il permet de gérer les collections, les prêts, les retours et la présence des usagers.

Le projet est séparé en 2 parties : une partie client (frontend) en TypeScript et une partie serveur (backend) en C#.

Il utilise ces technologies :

Pour la partie client :

- [React](https://en.wikipedia.org/wiki/React_(software)) : une bibliothèque pour construire l'interface
  utilisateur.

- [Bootstrap](https://en.wikipedia.org/wiki/Bootstrap_(front-end_framework)) : un framework de mise en page
  pour avoir une interface jolie et responsive.

Pour la partie serveur :

- [FastEndpoints](https://fast-endpoints.com/) : une bibliothèque pour créer des points d'entrée web de
  manière simple.

- [Entity Framework](https://en.wikipedia.org/wiki/Entity_Framework) : un framework open source qui relie
    le programme à la base de données.

- [MariaDB](https://en.wikipedia.org/wiki/MariaDB) : une base de données relationnelle pour stocker les données pour 
  la médiathèque
