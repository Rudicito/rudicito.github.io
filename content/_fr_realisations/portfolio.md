Ce portfolio est un site statique, hébergé par [GitHub](https://github.com/) gratuitement.
Il me permet de présenter mes réalisations et d'ajouter facilement de nouveaux projets.

Le fonctionnement repose sur des fichiers simples et des modèles réutilisables.
Cela garde une présentation régulière sur toutes les pages.

Le site utilise ces technologies :

- [Jekyll](https://en.wikipedia.org/wiki/Jekyll_(software)) : un générateur de site statique.

- [Markdown](https://en.wikipedia.org/wiki/Markdown) : un format de texte simple pour écrire les contenus.

- [Liquid](https://en.wikipedia.org/wiki/Liquid_(template_language)) : un moteur de modèles pour construire
  les pages automatiquement.
