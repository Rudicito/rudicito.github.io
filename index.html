Projets Des projets que j'ai réalisés dans mon temps libre et/ou pour l'école !

Tank télécommandé

Au lycée dans notre club informatique, nous avons réalisé un tank télécommandé. Il est composé d'un Arduino, 4 moteurs, d'une enceinte, et d'un module bluetooth. Le tank peut se controller à distance par un téléphone grâce au module bluetooth et au code en C++ de l'Arduino. Il peut également jouer de la musique avec l'enceinte.

DOOM sur la calculatrice HP Prime

C'est un port de DOOM, jeu sorti en 1993, écrit en Python, qui fonctionne sur la calculatrice HP Prime. Ce code est basée sur un projet de portage du jeu en Python sur PC (à la base en C). Il a fallu réadapter le code pour utiliser les librairies limité de la calculatrice.
